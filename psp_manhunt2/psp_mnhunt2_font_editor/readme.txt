Tile Molester 0.16 + bookmrks and palettes for editing PSP Manhunt 2 fonts

1. You'll need to decompress GLOBAL.TXD from Manhint 2 iso with with TDX_DeComp
https://github.com/1upus/psp_psv_modding_and_translations/tree/master/psp_manhunt2/TXD_DeComp

2. Open decompressed GLOBAL.TXD.u with TileMolester

3. Select font from bookmarks "Navigate->font1/2/3"

4. Select palette from "Palette->font1/2a/3"

5. Export-modify-import

6. Profit!!!
