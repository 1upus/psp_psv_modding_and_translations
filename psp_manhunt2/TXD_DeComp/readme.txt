TXDDeComp by Zalbard
======================

Simple console apps for decompress/recompress TXD files from Manhunt 2 (psp/ps2).


Using:
======================

Just drag TXD file to app.
DeCompressed TXD file will be named "%filename%.u", recompressed - "%filename%.p".
Apps was made for De/Compress GLOBAL.TXD and edit ingame fonts.
